### `/repos`

Get repository info.

#### README

* <https://docs.github.com/en/rest/repos/contents>

* Example: <https://api.github.com/repos/HackerNews/API/readme>

Use `atob()` when rendering Base64.

#### Get repository content

`/repos/{owner}/{repo}/contents/{path}`
